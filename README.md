## Password Locker
* This is a python application that allows the user to generate a password, store account credentials, view their credentials and save their account details

## By `Cecilia Barasa`

## Run 
`python3 account.py`


## Setup
* `Python3.6.9`
* `Pyperclip`
* `Termcolor`
